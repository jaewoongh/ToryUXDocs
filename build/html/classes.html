클래스 일람¶

CameraEffects¶

카메라 효과와 관련해 여러 스태틱 메소드들을 제공하는 클래스입니다. MonoBehaviour 클래스를 상속합니다.

Static Properties¶

Instance¶

public static CameraEffects Instance { get; }

CameraEffects 클래스는 싱글턴으로 동작합니다. 인스턴스를 참조하려면 이 프로퍼티를 사용하세요.

CurtainImage¶

public static Image CurtainImage { get; private set; }

페이드 효과에 사용되는 UI.Image입니다. 단순 사용을 위해서는 굳이 변경할 필요 없이 FadeIn과 FadeOut을 이용해 주세요.

기본 색상은 검정입니다. 색을 하양으로 바꾸고 FadeIn을 사용하면 카메라 플래시 효과로도 사용할 수 있습니다.

주석

커튼 이미지는 uielements.SettingsUI를 포함한 모든 ToryUX 요소 중 가장 앞에 그려집니다.

CurtainCanvasRenderer¶

public static CanvasRenderer CurtainCanvasRenderer { get; }

CurtainImage에 달린 CanvasRenderer입니다. 페이드 효과는 내부적으로 이 렌더러의 알파 값을 바꾸는 식으로 구현됩니다. 특별한 일이 없다면 굳이 손 댈 필요는 없습니다.

CurtainAlpha¶

public static float CurtainAlpha { get; set; }

CurtainImage에 달린 CanvasRenderer의 알파 값입니다. 현재 페이드 효과가 얼마나 진행되었는지 알기 위해, 또는 임의의 효과를 구현하기 위해 접근할 수 있습니다.

TranslucentLayer¶

public static TranslucentLayer TranslucentLayer { get; }

uielements.TitleUI나 uielements.ResultUI 등에서 UI 외 요소를 블러 처리하는데 사용되는 TranslucentLayer입니다.

BloomFlashIntensity¶

public static float BloomFlashIntensity { get; set; } = 5f

FlashBloom 효과에 적용될 블룸의 강도입니다. 기본으로 설정한 블룸 값에 이 값이 순간적으로 더해졌다 원래대로 돌아오는 식으로 동작합니다.

BloomSettings¶

public static BloomModel.Settings BloomSettings { get; set; }

필요할 경우 임의로 블룸 설정을 건드릴 수 있도록 열어 두었습니다.

VignetteIntensity¶

public static float VignetteIntensity { get; set; } = 0.45f

ShowVignetteEffect에 적용될 비네팅의 강도입니다. 기본으로 설정된 비네팅 값에서 이 값으로 변화하는 식으로 동작합니다. HideVignetteEffect를 호출하면 이 값에서 원래 설정돼있던 비네팅 값으로 변화합니다.

VignetteSettings¶

public static VignetteModel.Settings VignetteSettings { get; set; }

필요할 경우 임의로 비네팅 설정을 건드릴 수 있도록 열어 두었습니다.

Static Methods¶

FadeOut¶

public static void FadeOut(float duration = 1f, bool ignoreTimeScale = false)

페이드 아웃 효과를 냅니다. CurtainCanvasRenderer의 알파 값이 0에서 1로 변화합니다.

duration:전환 시간을 설정할 수 있습니다.
ignoreTimeScale:
 게임 시간을 사용할지 실제 시간을 사용할지 설정할 수 있습니다.

FadeIn¶

public static void FadeIn(float duration = 1f, bool ignoreTimeScale = false)

페이드 인 효과를 냅니다. CurtainCanvasRenderer의 알파 값이 1에서 0으로 변화합니다.

duration:전환 시간을 설정할 수 있습니다.
ignoreTimeScale:
 게임 시간을 사용할지 실제 시간을 사용할지 설정할 수 있습니다.

Fade¶

public static void Fade(float targetAlpha, float duration = 1f, bool ignoreTimeScale = false)

현재 CurtainCanvasRenderer의 알파 값에서 특정한 알파 값으로 변화하는 효과를 냅니다. 진행중인 페이드 인/아웃 효과를 도중에 취소하는 데에 유용합니다.

ShowTranslucentLayer¶

public static void ShowTranslucentLayer(float strength = -1f)

메인 카메라를 블러시키는 레이어를 활성화합니다.

strength:블러의 강도를 설정합니다. 숫자가 클수록 더욱 덩어리집니다. 음수 값을 건넬 경우 인스펙터 창에서 설정한 값으로(기본 30) 적용됩니다.

HideTranslucentLayer¶

public static void HideTranslucentLayer()

메인 카메라를 블러시키는 레이어를 비활성화합니다.

FlashBloom¶

public static void FlashBloom(float duration = 1f)

블룸 터뜨리는 효과를 냅니다. 화면 전환에 유용합니다.

duration:효과의 지속 시간을 설정할 수 있습니다.

주석

이 효과는 블룸 설정 중 intensity 값만 사용합니다. 세부 컨트롤이 필요하면 BloomSettings를 만져 주세요.

ShowVignetteEffect¶

public static void ShowVignetteEffect(float duration = 0.1f)

비네팅 효과를 냅니다. VignetteIntensity 값을 바꿔 정도를 조절할 수 있습니다.

duration:비네팅 효과가 나타나는 시간을 설정할 수 있습니다.

주석

이 효과는 비네팅 설정 중 intensity 값만 사용합니다. 세부 컨트롤이 필요하면 VignetteSettings를 만져 주세요.

HideVignetteEffect¶

public static void HideVignetteEffect(float duration = 0.1f)

비네팅 효과를 끕니다. ShowVignetteEffect를 호출하기 전에 사용하던 비네팅 값이 있다면 그 값으로 되돌아갑니다.

duration:비네팅 효과가 사라지는 시간을 설정할 수 있습니다.

UIOrientationSetter¶

ToryUX의 오리엔테이션과 관련해 몇 가지 편의 기능을 제공하는 스크립트입니다. MonoCollection<T> 클래스를 상속합니다.

Static Properties¶

CurrentOrientation¶

public static UIOrientation CurrentOrientation { get; }

현재 UIOrientation 값을 확인할 수 있습니다.

주석

public enum UIOrientation
{
   Unknown,
   Landscape,
   LandscapeUpsideDown,
   PortraitLeft,
   PortraitRight
}

ReferenceResolution¶

public static Vector2 ReferenceResolution { get; }

UIOrientationSetter의 부모가 가진 CanvasScaler의 referenceResolution 값을 돌려줍니다.

주의

씬 안에 UIOrientationSetter가 둘 이상 존재할 경우 (예컨대 캔버스 베이스 게임의 경우) 모든 부모 CanvasScaler의 referenceResolution 값이 같아야 정상 동작합니다.

만약 제대로 설정돼 있지 않더라도 아무런 오류를 띄우지 않으니 주의해 주세요.

Properties¶

Orientation¶

public UIOrientation Orientation { get; set; }

현재 UIOrientation 값을 읽거나 지정할 수 있습니다.

주의

이 프로퍼티는 개별 UIOrientationSetter 인스턴스의 값입니다.

씬 안에 UIOrientationSetter가 둘 이상 존재할 경우 (예컨대 캔버스 베이스 게임의 경우) 이 프로퍼티에 수동으로 값을 넣어도 다른 UIOrientationSetter에는 적용되지 않으므로, SetOrientation 스태틱 메소드를 사용해 주세요.

Static Methods¶

SetOrientation¶

public static void SetOrientation(UIOrientation newOrientation)

새 UIOrientation 값으로 설정합니다. 씬 안에 UIOrientationSetter가 둘 이상 존재할 경우 모든 인스턴스에 적용됩니다.

newOrientation:새로 사용할 UIOrientation 값을 지정합니다.

FlipOrientation¶

public static void FlipOrientation()

화면을 뒤집습니다. 씬 안에 UIOrientationSetter가 둘 이상 존재할 경우 모든 인스턴스에 적용됩니다.
