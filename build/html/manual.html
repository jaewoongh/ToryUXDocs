매뉴얼¶

불러오기¶

  1. 유니티 버전 2017 이상에서 동작합니다.
  2. 에셋 스토어에서 Post Processing Stack 을 임포트합니다.
  3. 최신 버전의 ToryUX 유니티 패키지를 실행해 프로젝트에 얹습니다.

주의

블러 효과를 위해 Translucent Image 에셋 번들에 의존합니다.
모바일에서도 퍼포먼스 부담 없이 제대로 큰 덩어리 블러 주는 보기 드문 에셋이니 참고 씁시다.

시작하기¶

  1. Project 탭에서 ToryUX/Prefabs/_ToryUX.prefab을 Hierarchy 탭으로 끌어와 씬에 얹습니다.

    주석

    앞으로 모든 캔버스 요소들은 전부 Hierarchy 탭 _ToryUX/Canvas/ToryUX밑에 넣어 주세요. (→ 예외)

  2. 끌어온 _ToryUX 게임오브젝트를 선택한 뒤 Inspector 탭에서 CameraEffects 컴포넌트를 찾아 Main Camera 필드에 메인 카메라를 끌어 넣어 줍니다.

  3. Project 탭 ToryUX/Prefabs 폴더 안 필요한 UI 요소들을 Hierarchy 탭 _ToryUX/Canvas/ToryUX 밑으로 끌어 넣습니다.

    주의

    ToryUX 관련 많은 스크립트들이 MonoBehaviour.Awake() 콜에 의존해 초기화합니다. 실행 전 임의로 게임오브젝트들을 비활성화할 경우 정상 동작하지 않을 수 있습니다.

캔버스 베이스 게임의 경우¶

프로젝트에서 사용할 모든 캔버스 요소들을 원칙적으로 Hierarchy상 _ToryUX/Canvas/ToryUX 밑으로 들어가야 합니다. 하지만 버거 타워나 캔디 몬스터처럼 게임 자체가 캔버스에서 동작하는 경우 UI 관련 캔버스 요소와 게임 관련 캔버스 요소를 구별할 필요가 있습니다.

  1. Project 탭 ToryUX/Prefabs/GameCanvas.prefab을 Hierarchy 탭으로 끌어다 넣습니다.
  2. 게임 관련 캔버스 요소들은 모두 Hierarchy GameCanvas/OrientationSetter 밑으로 만들기 시작해 주세요.

ToryUX나 기타 UI 관련 캔버스 요소들은 여전히 _ToryUX/Canvas/ToryUX 아래로 넣어 사용하시면 됩니다.
